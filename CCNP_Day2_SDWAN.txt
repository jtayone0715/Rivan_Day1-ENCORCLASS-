4. Organization Name must match at the certificate

5. in Edge Router, 
- vshell to go linux from Cisco (to get the copied certificate)
- create a certificate using vi and put it in /home/admin/pkicerts/(created file is rivanpki.ca)
- command to install the cert (go back to cisco from linux)
- request root-cert-chain install /home/admin/pkicerts/rivanpki.ca

7. in Edge Router, command to create a certificate csr for the root CA, use vi
request csr upload /home/admin/pkicerts/min.csr

8. to sign the certificate csr by the root CA (Another Server, PKI Server)
crypto pki server rivanpki request pkcs10 terminal
(paste the csr)
(granted csr is generated)
9. in Edge Router, command to enter the granted cert
